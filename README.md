**MIMIC**은 **BitTorrent** 및 [WebTorrent](https://github.com/webtorrent/webtorrent)를 지원하는 스트리밍 토렌트 클라이언트 입니다.

> 60%정도 진행된 상태입니다

### 설치

```bash
git clone
cd mimic
yarn # or npm install
yarn run dev # or npm run dev
```
